# SitoFSL NTT Data

Insieme, io ([dico221](https://github.com/dico221)) e [Moneta Gianluca](https://github.com/MonetaGianluca) abbiamo creato un sito web, da zero, in cui raccontiamo le varie attività svolte durante il periodo di FSL in NTT Data *(nella sede di Milano)*.

Abbiamo iniziato con un design semplice su [figma](https://www.figma.com/). Dopodichè l'abbiamo presentato davanti a vari designers presso l'azienda.

Siamo riusciti, con un po' di aiuto da parte dell' Intelligenza Artificiale, a realizzare il nostro sito nel modo più genuino possibile.
